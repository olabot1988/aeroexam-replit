Create an aviation maintenance examination app. Current regulations require all mechanics to complete a hand written examination annually. this app will serve as the platform for testing. The app will have a basic homepage with a “Begin Test” button located in the middle of the page. When the user clicks the button, the app will transition to the admin portion of the exam. the admin portion will prompt the mechanic to input their first name, last name, maintenance level(ML0, ML1, ML2, ML3, ML4), the type of exam(Annual Exam, Technical Inspector Exam, CDR Progression Written Exam). Next, they will be prompt to enter a 6-15 character password. The password and their last name will be used to continue the exam at a later time if required. All fields in the admin portion require an entry. The mechanic will not be able to move forward if a field is left blank. Once the mechanic selects “Next”, they will be taken to a page that informs them, they are about to begin a 50-questions [blank] test and that the minimum passing score is 90%. replace blank with the type of test they selected in the admin portion. for example if someone selected Annual Exam, then the page should say: You are about to take a 50-question annual exam. The mechanic will have the option to “begin test”, “go back” or “return home”. once the mechanic selects “begin test”, they will begin taking the exam. the exam will consist of multiple choice or true/false questions. the difficulty of the exam will depend on the maintenance level of the mechanic. mechanics taking a CDR Progression Written Exam will take an exam meant for the next higher ML. for example a ML2 mechanic will take a ML3 exam. Technical Inspector exam will be ML3 difficulty. The exam will include a progress bar at the top for the user to track their progress. Their will be a “Leave Exam” button that will allow the user to save their progress and continue at a later time. If the user selects this button, a prompt will pop up asking them if they are sure they want to exit the exam. If they select yes, another prompt will pop up reminding them to use their last name and password to continue the exam. the examination will also include a “Quit Test” button that will return to the home page when pressed. when the button is clicked, a prompt will open asking them if they are sure they want to quit and warning them that quitting the test won’t save their progress. Once they answer the last question, a “submit test” button will populate. clicking this button will transition the mechanic to the results page which will show them their score and all questions answered. questions answered correctly will have a green check mark while questions answered incorrectly will contain a red x. The user will have the option of printing the exam for records keeping.This app should be user friendly with a aviation themed color scheme. This app should be worthy of a 4.8 Google Play rating.
